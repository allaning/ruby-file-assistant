FileAssistant

Ruby File Assistant (RFA) helps automate mundane file management tasks. Create configuration files to instruct RFA to perform tasks such as deleting or renaming files based on specified patterns.

Public Instance Methods

delete_files( patterns ) click to toggle source

Delete files matching specified patterns. Patterns must be specified in an array.

# File bin/file_assistant.rb, line 22
def delete_files( patterns )
  patterns.each do |pattern|
    found_files = Dir.glob( "#{pattern}" )
    found_files.each do |found_file|
      puts "Deleting file: #{found_file}"
      File.delete( found_file )
    end
  end
end
files_to_delete() click to toggle source

If the config file containing list of patterns to delete exists, open the file and return its contents as an array

# File bin/file_assistant.rb, line 10
def files_to_delete
  if File.exists?( FileAssistantConfig.to_delete )
    config_file = File.open( FileAssistantConfig.to_delete, "r")
    contents = []
    config_file.each { |line| contents << line.chomp }
    config_file.close
    return contents
  end
end
